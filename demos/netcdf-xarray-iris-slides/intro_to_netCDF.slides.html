netCDFs in Python¶

netCDF: Network Common Data Form¶

  • Self describing
  • Sharable
  • Scalable
  • Appendable

Uses:

  • Climate model output
  • Satalite products
  • Meteorological/hydorlogical forecasts
  • Reanalysis products
  • Data with high dimension or spatiotemporal grids

Python tools¶

  • iris
  • xarray
  • others such as netcdf4, cf, geopandas

xarray and iris¶

iris:

  • metoffice
  • specific (netCDF, GRIB, PP)
  • rigorously cf compliant

xarray:

  • a more generic data structure (hd5, json etc...)
  • isn't strict on cf compliance
  • based on pandas

The cube¶

Contents of a cube¶

  • The data of any number of dimensions
  • The dimensional coordinates
  • Any auxillary coordinates
  • Metadata

Dimensional coordinates¶

Coords that describe one dimension/axis of the data:

  • latitude, time, ensemble number etc
  • Are fully descriptive (have units and abide by CF conventions)

Auxillary coordinates¶

Coords which aren't necessary but may be useful:

  • Day of the year
  • A coordinate factory: a way of building a coord from an existing coord e.g. hybrid height
